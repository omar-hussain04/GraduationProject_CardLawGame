# 🃏 Cardlaw - 2D Platformer Game

## 📖 About
Cardlaw is a 2D Pixel Art 16-Bit platformer built with the **Godot Engine**.
The player must save four kingdoms from the "Hammer Joker", each kingdom
representing a card suit (♠ ♥ ♦ ♣).

## 🎮 Features
- Pixel Art 16-Bit visual style
- Multi-level platformer with boss fights
- AI-assisted sound generation via Ludo.ai
- Supports keyboard, touch, and controller input
- Available on Desktop, Mobile, and Web

## 🛠️ Built With
- Godot Engine (GDScript)
- Figma & Illustrator (UI/UX Design)
- Ludo.ai (Sound Integration)
